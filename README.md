VolumCalc
=========

Kalkulator for å regne ut volum a prisme, trekant prisme, sylinder og kule.

Instalasjon:
  1.  Ha nyeste versjon av python instalert på maskinen.
  2.  Dobbelklikk på .py filen og enjoy.
